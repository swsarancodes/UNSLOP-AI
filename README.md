# unslopper

Agent skill that stops AI from writing like an AI.

Strips sycophancy, hype words, contrast patterns, em dashes, and other tells that make AI-generated text easy to spot. Install as a skill for Claude Code, Codex, Cursor, and other agent tools.

## Install

```bash
npx skillpm install unslopper
```

Then invoke with `/avoid-ai-speak` or use it automatically when writing prose.
